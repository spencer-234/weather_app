About

This is a full stack weather app. The frontend is made with html, css, and javascript. The backend (server) is made using Node.js. The backend fetches data from the Free Weather Api to retrieve accurate weather information for the area based on the zip code enetered by the user. This is my first mini project under the Institute of data where we showcase what we've learned about html, css and javascript so far.
